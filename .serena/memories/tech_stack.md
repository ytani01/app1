# Tech Stack
This project is built using Python. It leverages `uv` for package management and build processes.

## Core Dependencies:
- `click`: For creating beautiful command line interfaces.
- `opencv-python`: For displaying the game on environments without `pi0disp`.
- `pi0disp`: A local dependency for displaying the game on Raspberry Pi connected LCDs.
- `pigpio`: A Python module for Raspberry Pi GPIO.

## Development Dependencies:
- `mypy`: A static type checker for Python.
- `pytest`: A testing framework for Python.
- `ruff`: An extremely fast Python linter and formatter.

## Python Version:
- Requires Python 3.13 or newer.
