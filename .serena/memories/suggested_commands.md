# Suggested Commands

## Project Setup
- `uv install`: Install all dependencies (including dev dependencies)
- `uv run app1`: Run the game (entrypoint `app1:main`)

## Development Commands
- `uv run pytest`: Run tests
- `uv run ruff check .`: Run linter
- `uv run ruff format .`: Run formatter
- `uv run mypy src/`: Run type checker

## General Utilities
- `git status`: Check the status of the git repository
- `git add <file>`: Stage changes for commit
- `git commit -m "Message"`: Commit staged changes
- `git push`: Push committed changes to remote
- `ls -F`: List files and directories
- `cd <dir>`: Change directory
- `grep <pattern> <file>`: Search for a pattern in a file
- `find . -name "<pattern>"`: Find files by name
